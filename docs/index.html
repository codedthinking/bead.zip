Reproducible Research.
Simplified.

Bead packages your computational workflows into self-contained, versioned units with automatic dependency tracking. No more "what data did we use?" questions.

Get Started View on GitHub
Create reproducible workflows in seconds 
# Create a new computational unit
$ bead new german-cities
$ cd german-cities

# Add dependencies
$ bead input add population-data

# Run your analysis
$ python analyze.py

# Save immutable snapshot
$ bead save results
